## 📌 Project Overview

This is a comprehensive web development project created for the Advanced Web Design and Development course (BIT3208). The project demonstrates the implementation of modern web technologies including **PHP**, **JavaScript**, **MySQL**, **HTML5**, and **CSS3** to build a fully functional web application with user authentication, form validation, database integration, and dynamic content manipulation.

### 🎯 Project Objectives

- Set up a complete local development environment (XAMPP/Localhost)
- Design professional wireframes and GUI layouts
- Implement JavaScript for client-side interactivity
- Create server-side components using PHP
- Develop database-driven CRUD operations
- Build a secure authentication system
- Practice DOM manipulation techniques
